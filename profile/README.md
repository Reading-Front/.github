##  스터디 진행 방식
- 매주 1챕터 읽고 요약 및 토론
- GitHub PR로 정리글 제출 (스터디 진행 시작 전까지)
- Github Issues로 각 챕터별 문제 제출 (토요일 자정 전까지)
- 결석 시 최소 하루 전 공유 필수!
- 서로 질문과 피드백 주고받기

## 🔫 그라운드 룰
- PR은 마감일 전까지 제출
- 적극적인 피드백, 서로 존중 (오탈자, 잘못 기재된 부분 리뷰하기)
- 결석 시 미리 공유
- 챕터별 문제 출제는 이슈로 공유
- 문제 출제는 스터디 하루 전 토요일 자정까지 업로드
